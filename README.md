# 🔐 Login System (Python + SQL)

Hi! 👋  
This is a simple login system I built using Python and SQLite as part of my learning journey after completing 12th grade.

I wanted to understand how real applications handle user authentication, so I created this small project that allows users to register, log in, and delete their accounts.

---

## 🚀 Features

- 📝 User Registration  
- 🔑 User Login  
- 🗑️ Delete User  
- 🔒 Password hashing for basic security  
- 📋 Menu-driven interface (easy to use)

---

## 🛠️ Technologies Used

- Python  
- SQLite (using `sqlite3`)  

---

## 📱 Built On

I developed this entire project on my mobile phone using Pydroid 3, which helped me practice coding without a laptop.

---

## 💡 What I Learned

- How databases work with Python  
- Basic SQL operations (INSERT, SELECT, DELETE)  
- How login/authentication systems function  
- Importance of securing passwords using hashing  
- Writing structured and readable code  

---

## ▶️ How to Run

1. Download or clone this repository  
2. Open the file in Pydroid 3 (or any Python IDE)  
3. Run the program  
4. Use the menu options to register, login, or delete users
